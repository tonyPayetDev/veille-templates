Prérequis techniques

Tout est gratuit sauf l'API vocale. Compte 15 minutes d'installation, une seule fois.

  • Claude Code — CLI, app desktop ou extension IDE. C'est lui qui exécute le skill.
  • Node.js 20+ — requis par HyperFrames.
  • HyperFrames — le moteur de rendu vidéo (compositions HTML → MP4). Installé par npx hyperframes init, puis npx hyperframes doctor pour vérifier l'environnement.
  • Chrome / Chromium — HyperFrames rend les frames dans un navigateur headless. npx hyperframes browser install s'en occupe.
  • ffmpeg — assemblage audio/vidéo, détourage de l'avatar, encodage final.
  • Une clé API WaveSpeed — pour la voix clonée. À toi, facturée à l'usage, ≈ 0,50 € par vidéo.
  • Tes deux assets — voice-reference-clean.wav (30 s de ta voix) et avatar.mp4 (45 s face caméra). Le skill dit précisément comment les enregistrer.

Machine : 8 Go de RAM minimum, 16 Go confortable. Un rendu de 30 s prend 3 à 8 minutes selon la machine.
