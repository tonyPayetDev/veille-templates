Découvrir ShortForge
Installation
  1. Télécharge l'archive après l'achat, dézippe-la.
  2. Copie le dossier shortforge/ dans .claude/skills/ de ton projet (ou ~/.claude/skills/ pour l'avoir partout).
  3. Relance Claude Code, puis tape /shortforge — le skill doit apparaître dans la liste.
  4. Renseigne ta clé WAVESPEED_API_KEY dans .claude/settings.json, section env.
  5. Prérequis : Node 20+, npx hyperframes init puis npx hyperframes doctor, Chrome headless (npx hyperframes browser install) et ffmpeg.
La voix (clonage)
  1. Enregistre 20 à 40 s de ta voix, au calme, sans musique ni respiration coupée. Format WAV mono, 44.1 kHz.
  2. Range le fichier dans assets/voice-reference-clean.wav du projet.
  3. Le skill appelle le modèle wavespeed-ai/qwen3-tts/voice-clone avec ce fichier comme référence.
  4. Débit réel ≈ 3,3 mots/seconde : vise 100-115 mots pour une vidéo de 25-35 s. Si c'est trop long, réécris le script et régénère — n'accélère jamais l'audio, ça s'entend.
  5. Les sigles et mots-clés du CTA se prononcent mal : écris-les phonétiquement dans le script vocal (le texte à l'écran, lui, reste correct).
L'avatar fixe
  1. Filme une seule prise de 30-60 s, cadrage buste, regard caméra, fond uni (vert ou mur clair), lumière frontale.
  2. Place le fichier dans assets/avatar.mp4. Le skill le boucle sur toute la durée de la vidéo — pas besoin de retourner filmer par script.
  3. Fond vert : le détourage se fait au filtre geq (pas chromakey), avec correction du canal vert pour supprimer le liseré.
  4. L'avatar est ancré en bas de cadre et ne doit jamais chevaucher les captions ni le texte animé.
  5. Coupe la première seconde si tu clignes des yeux au démarrage — un clignement sur la frame d'ouverture casse le hook.
