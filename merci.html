1 · Installation

Deux minutes, une seule fois.

  1. Dézippe l'archive.
  2. Copie le dossier shortforge/ dans .claude/skills/ de ton projet — ou dans ~/.claude/skills/ pour l'avoir sur tous tes projets.
  3. Relance Claude Code, puis tape /shortforge : le skill doit apparaître dans la liste.
  4. Renseigne ta clé WAVESPEED_API_KEY dans .claude/settings.json, section env.
  5. Prérequis : Node 20+, puis npx hyperframes init et npx hyperframes doctor, Chrome headless (npx hyperframes browser install) et ffmpeg.

2 · Ta voix

Le clonage vocal ne vaut que ce que vaut ton échantillon.

  1. Enregistre 20 à 40 secondes de ta voix, au calme, sans musique ni respiration coupée. WAV mono, 44.1 kHz.
  2. Range le fichier dans assets/voice-reference-clean.wav de ton projet.
  3. Le skill appelle le modèle wavespeed-ai/qwen3-tts/voice-clone avec ce fichier comme référence.
  4. Débit réel : ≈ 3,3 mots par seconde. Vise 100 à 115 mots pour une vidéo de 25-35 s. Si c'est trop long, réécris le script et régénère — n'accélère jamais l'audio, ça s'entend.
  5. Les sigles et les mots-clés de CTA se prononcent mal : écris-les phonétiquement dans le script vocal. Le texte à l'écran, lui, reste correct.

3 · Ton avatar

Un seul tournage, réutilisé sur toutes tes vidéos.

  1. Filme une seule prise de 30 à 60 secondes : cadrage buste, regard caméra, fond uni (vert ou mur clair), lumière frontale.
  2. Place le fichier dans assets/avatar.mp4. Le skill le boucle sur toute la durée de la vidéo — tu ne refilmes pas par script.
  3. Sur fond vert, le détourage se fait au filtre geq (pas chromakey), avec correction du canal vert pour supprimer le liseré.
  4. L'avatar est ancré en bas de cadre et ne doit jamais chevaucher les captions ni le texte animé.
  5. Coupe la première seconde si tu clignes des yeux au démarrage : un clignement sur la frame d'ouverture casse le hook.
Le README.md de l'archive reprend tout ça en détail, avec les commandes exactes. Commence par lui.
